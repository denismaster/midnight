[Получение профессии классом Орк-Маг]:
Когда Orc Mystic достигает 19 уровня, персонажу разрешается начать прохождение заданий для получения первой профессии. По успешному окончанию квеста и достижении 20 уровня, персонаж может получить профессию Orc Shaman. После получения профессии, персонаж может обладать более мощными и разнообразными навыками.
Чтобы стать Orc Shaman поговорите c Tataru Zu Hestui в деревне Орков.
